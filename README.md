- PyMuPDF
- pdfplumber
- pandas
- openpyxl
- Streamlit

## Run locally

```bash
pip install -r requirements.txt
streamlit run app.py
```

## Deploy on Streamlit Cloud

Upload these files to the GitHub repository root:

- app.py
- requirements.txt
- README.md

Then deploy with:

```text
Main file path: app.py
```
